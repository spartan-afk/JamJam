# JAMJAM 🎵

**Turn Spotify listening into a public hangout — join live rooms, discover music with strangers, save the vibe as a playlist.**

JAMJAM is a social listening platform built on top of the Spotify Web Playback SDK and Supabase Realtime. It allows users to host public rooms, play music, and let anyone drop in to vibe and chat in real-time.

## Features
- **Live Sync:** Music playback, album art, and waveforms sync instantly across all listeners.
- **Public Discovery:** Browse active rooms by "vibe" (Chill, Party, Study, Discover, Late Night).
- **Real-time Chat:** Chat with others in the room instantly.
- **Save the Vibe:** Automatically logs every track played so the host can export the session to a Spotify playlist with one click.
- **Your Rooms & Friends:** See your past rooms and users you've recently jammed with.

## Tech Stack
- **Frontend:** Next.js (App Router), TailwindCSS, React
- **Backend:** Next.js Route Handlers
- **Database & Realtime:** Supabase (PostgreSQL)
- **Audio & Auth:** Spotify Web API & Web Playback SDK
